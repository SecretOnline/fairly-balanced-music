# Fairly Balanced Music

A resource pack that rebalances the way music plays in Minecraft.

Check [MODRINTH.md](./MODRINTH.md) for the actual pack description.

## Developing

This pack is mostly automated, checking for new music every update, so there shouldn't be much to worry about. All of this is in the `scripts` folder as Typescript files. Try and avoid adding new dependencies to the project.
