# Fairly Balanced Music

I like the classic Minecraft soundtrack. I also like the newer additions too. I want to hear it all no matter where I am in the world, so let's throw it all into one great big list and let the game select from it at random. No more biome-specific music, no more weighting of certain tracks that drown out the rest. Just all of the music, selected fairly, and fairly balanced.

## Changes to music

### Overworld

Every biome has the same track list. This list is made from:

- The default track list.
- Each biome-specific track list.
- The Creative mode exclusive tracks.

All of the heavy weighting given to new tracks has been removed, so every track in the list has the same chance of playing.

Things that have not been changed:

- The music that plays in the Deep Dark biome (Ancestry and Deeper) has been left untouched. It didn't feel right to add the other music in to the dark and spooky place.
- The underwater music tracks added in 1.13 are still the only ones that play underwater.

### Nether

All of the existing Nether music plays in all biomes, except for the Warped Forest. The Warped Forest does not play any music.

### The End

The music in The End has not been changed.
